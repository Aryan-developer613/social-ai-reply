# Project

Projects, brand profiles, prompt templates, and campaign configuration.

## Project

### Fields
- `id` - Unique identifier
- `workspace_id` - Parent workspace
- `name` - Project name
- `slug` - URL-friendly identifier
- `description` - Project description
- `status` - active/archived
- `created_at` - Creation time
- `updated_at` - Last update time

### Purpose
- Represents a brand or campaign
- Contains its own opportunities
- Has independent settings
- Scoped to workspace

### Lifecycle
1. Create project in workspace
2. Configure brand profile
3. Set up prompts and keywords
4. Run agents to discover opportunities
5. Manage opportunities and drafts

## Brand Profile

### Fields
- `id` - Unique identifier
- `project_id` - Parent project
- `website_url` - Company website
- `brand_name` - Company name
- `summary` - Business description
- `products` - Product list
- `audience` - Target audience
- `voice_notes` - Writing style
- `call_to_action` - CTA style
- `created_at` - Creation time
- `updated_at` - Last update time

### Purpose
- Stores brand intelligence
- Generated by Brand Brain
- Used by all agents
- Guides content generation

### Intelligence sources
- Website crawling and analysis
- Manual input and refinement
- LLM-powered extraction
- Keyword expansion

## Prompt Template

### Fields
- `id` - Unique identifier
- `project_id` - Parent project
- `name` - Template name
- `prompt_type` - reply/post
- `instructions` - Template content
- `is_default` - Default template flag
- `created_at` - Creation time
- `updated_at` - Last update time

### Purpose
- Custom instructions for LLM generation
- Different templates for replies vs posts
- Project-specific customization
- Reusable across opportunities

### Template types
- **Reply templates** - For responding to opportunities
- **Post templates** - For creating original content

## Usage patterns

### Creating a project
```python
from app.db.tables.projects import create_project

project = create_project(supabase, {
    "workspace_id": workspace_id,
    "name": "My Brand",
    "slug": "my-brand"
})
```

### Getting brand profile
```python
from app.db.tables.projects import get_brand_profile_for_project

brand = get_brand_profile_for_project(supabase, project_id)
```

### Using prompts
```python
from app.services.product.copilot import ProductCopilot

copilot = ProductCopilot()
draft = copilot.generate_reply(
    opportunity=opportunity,
    brand=brand,
    prompts=prompts
)
```

## Database tables

- `projects` - Project records
- `brand_profiles` - Brand intelligence
- `prompt_templates` - Custom instructions

## API endpoints

- `GET /v1/projects` - List projects
- `POST /v1/projects` - Create project
- `GET /v1/projects/{id}` - Get project
- `PUT /v1/projects/{id}` - Update project
- `GET /v1/projects/{id}/brand` - Get brand profile
- `PUT /v1/projects/{id}/brand` - Update brand profile

## Relationships

### To workspace
- Projects belong to workspaces
- Workspace manages billing and users
- Project settings are independent

### To opportunities
- Projects discover opportunities
- Opportunities are scoped to projects
- Multiple projects per workspace

### To agents
- Agents run per project
- Brand profile feeds agents
- Prompts customize generation
